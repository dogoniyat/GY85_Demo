# GY85_Demo
Test programs for the GY-85 9DOF board (ITG3200) / Magnetometer (HMC5883) / Accelerometer (ADXL345)
This projects contains 2 parts:
- The Arduino part, where the GY085 chip is hooked to an Arduino Nano compatible board
- The Processing script gathering the data collected by the Arduino and sent to the computer through USB
  and rendering it into fancy dials and a 3D openGL model
